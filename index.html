New core courses requiring creation
Create | Core
CPSC 454
Large Language Models & Generative AI
Prereq: CPSC 393
Transformers, fine-tuning (LoRA/QLoRA), RLHF, RAG, prompt engineering, and responsible deployment. Students fine-tune open-source LLMs and build end-to-end generative AI applications.
Create | Core
CPSC 455
Reinforcement Learning
Prereq: CPSC 393, MATH 203
MDPs, Q-learning, DQN, policy gradients (PPO), actor-critic architectures, and model-based RL. Applications span game playing, robotics simulation, and RLHF for language models.
Create | Core
CPSC 460
Distributed AI & MLOps
Prereq: CPSC 454
Production AI engineering: distributed training (DDP, FSDP), model serving (vLLM, Triton), experiment tracking (MLflow, W&B), Docker/Kubernetes, CI/CD for ML, and model drift detection.
Create | Core | Piloting Fall 2025
CPSC 468
Agentic AI Systems
Prereq: CPSC 454 | Being offered as CPSC 370, Fall 2025
LLM-based agent architectures, tool use, function calling, multi-agent coordination, planning and memory systems, and safety for autonomous AI. Students design and evaluate agentic pipelines.
Existing course added as new requirement
Add Req | Core
CPSC 445
High Performance Computing
Prereq: CPSC 350 | Moved from elective to core
OpenMP, MPI, parallel performance evaluation. Provides the parallelism foundations students need before CPSC 450 (GPU Programming & CUDA). No course changes needed.
New elective courses requiring creation
Create | Elective
CPSC 450
GPU Programming & CUDA
Prereq: CPSC 445, CPSC 231
Massively parallel programming on NVIDIA GPUs using CUDA C/C++. Thread/block/grid hierarchy, memory hierarchy, cuBLAS/cuDNN, and PyTorch CUDA extensions. Students implement and benchmark GPU kernels.
Create | Elective | Cross-list CPSC 542
CPSC 470
Computer Vision
Prereq: CPSC 393
Neural network-based deep learning methods for computer vision. CNNs, object detection, segmentation, image generation (GANs, diffusion models), and multi-modal vision-language models. Cross-lists with CPSC 542 (graduate). Prereq replaces MS admission requirement.
Create | Elective | Cross-list CPSC 543
CPSC 480
Natural Language Processing
Prereq: CPSC 393
Data-driven NLP methods from industry and academia: linguistics, tokenization, embeddings, sequence models, and modern transformer-based approaches. Cross-lists with CPSC 543 (graduate). Prereq replaces MS admission requirement.
Create | Elective | Or align with CPSC 370 Fall 2026
CPSC 475
AI Ethics, Law & Policy
Prereq: CPSC 285
Algorithmic bias, fairness metrics, explainability (LIME, SHAP), differential privacy, EU AI Act, US regulatory frameworks, and responsible AI practices. Case studies from healthcare, criminal justice, and hiring.

(2) CPSC 470 and CPSC 480 cross-list with CPSC 542 and CPSC 543 - MS admission requirement replaced by CPSC 393. (3) CPSC 475 may align with CPSC 370 "Responsible AI" sections planned for Fall 2026. (4) MATH 360 remains available as a free elective for students who want additional probability background.
